You can execute the script with the following command : 
```
python3 src/utils/eligibility_check_hm.py

```
When the script is executed, the following steps are performed : 

- The script will ask for the path of the csv file
- The script will read the csv file
- The script will check the eligibility of the vehicles using the API of the HM
- The script will save the result in a csv file and add a column called "Activation" with the result
